# 🌿 MonsteraOpti

> **Advanced Gaming Optimization Suite** — Zoptymalizuj swój system, śledź swoje postępy i wejdź w tryb Beast Mode.

---

## 🚀 O Projekcie

**MonsteraOpti** to potężne narzędzie stworzone dla graczy, którzy chcą wycisnąć ze swojego sprzętu 100%. Łączy w sobie zaawansowany monitoring systemu, śledzenie czasu gry oraz inteligentną optymalizację procesów w jednym, ultra-nowoczesnym interfejsie.

## ✨ Kluczowe Funkcje

- 📊 **Live System Stats** — Monitoruj zużycie CPU, GPU i RAM w czasie rzeczywistym z pięknymi wykresami.
- 🎮 **Game Library** — Zarządzaj swoimi grami z platform Steam, Epic Games oraz PC w jednym miejscu.
- ⏱️ **Playtime Tracker** — Automatyczne śledzenie czasu spędzonego w Twoich ulubionych tytułach.
- 🛡️ **Bloatware Remover** — Szybkie czyszczenie systemu ze zbędnych procesów spowalniających rozgrywkę.
- ⚡ **Beast Mode** — Jedno kliknięcie, by przełączyć system w tryb maksymalnej wydajności z dedykowanym motywem wizualnym.

## 🔥 Beast Mode

Aktywuj **Beast Mode**, aby uwolnić pełną moc swojego komputera. Aplikacja automatycznie dostosowuje priorytety procesów i zmienia interfejs na agresywny, czerwony motyw, sygnalizujący gotowość do walki.
